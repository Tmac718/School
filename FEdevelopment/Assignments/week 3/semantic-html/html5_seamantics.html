Articles for content

Article Tag

The < article > element represents an independent article in a web page. It should only wrap content that can be plucked out of your page and distributed in a completely different context. We can use <article> to mark the main content of the page as a self-contained unit.

Aside Tag

The <aside> tag makes it easy to design the page and it enhances the clarity of HTML document. It let us easily recognize the main text and subordinate text. In both the time <div> and <aside> need CSS to specific design. The <aside> tag supports Global attributes and Event attributes in HTML. This description of aside tags is written entirely in a paragraph tag in the article section of the websites content. The <aside> HTML element represents a portion of a document whose content is only indirectly related to the document's main content. Asides are frequently presented as sidebars or call-out boxes.

Nothing Special

The <aside> tag is new in HTML5. This tag does not render as anything special in a browser you have to use CSS for that.

Highlight

Aside is used to sum up the object of the main content. So we briefly describe the content like a highlighter.

Author Info and Links

The Aside tag is often used to contain Author information, links and related content.
